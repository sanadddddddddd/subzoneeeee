# Subzone Warranty Portal

نسخة أولية لبوابة ضمان مبنية لفكرة:

- `QR` ثابت في كل البطاقات.
- العميل يفتح `portal.html`.
- يدخل `رقم الهاتف + كود الضمان`.
- تظهر له البيانات المسجلة له فقط.

## الصفحات

- `index.html`: الصفحة الرئيسية.
- `portal.html`: بوابة العميل.
- `admin.html`: لوحة إدارة لإضافة العملاء وتعديلهم.
- `card.html`: بطاقة ضمان قابلة للطباعة مع مكان مخصص للـ QR الثابت.

## طريقة الاستخدام

1. افتح `admin.html`.
2. أضف بيانات العميل وقت البيع.
3. سيظهر له `كود ضمان` يمكن كتابته على البطاقة.
4. ارفع الملفات على استضافة.
5. حوّل رابط `portal.html` إلى QR واحد ثابت.
6. اطبع نفس البطاقة لكل العملاء.

## التخزين

- البيانات تُحفظ محليًا داخل المتصفح عبر `localStorage`.
- هذه مناسبة كنموذج أولي أو عرض فكرة.
- إذا تبي نسخة إنتاجية حقيقية، الخطوة التالية تكون ربطها بقاعدة بيانات حقيقية ولوحة دخول آمنة.

## الشعار

- الملف الحالي: `assets/subzone-mark.svg`
- هذا شعار واجهة مؤقت مستلهم من هوية `Subzone`.
- إذا عندك الشعار الرسمي بصيغة `PNG` أو `SVG`، استبدله بنفس الاسم ليظهر مباشرة في كل الصفحات.
